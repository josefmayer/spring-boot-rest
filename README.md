## Spring Boot REST

### Technologies
Spring Boot, REST


### Steps
##### Build and Run Application
via spring-boot-maven-plugin: <br />
*mvn spring-boot:run*

##### Build jar, Run jar
*mvn package* <br />
*java -jar target/jar-name.jar*

##### Access Application
*http://localhost:8080/hello* <br />
*http://localhost:8080/reverse?input=mytext*
